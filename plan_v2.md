# 球面世界模型：在 LeWM 基础上探索球面表征的最小实验方案

## Motivation

Joint-Embedding Predictive Architecture (JEPA) 系列的世界模型最近取得了显著进展，但仍存在一些值得探讨的问题。LeWorldModel (LeWM) 作为一个端到端、从像素直接训练的 JEPA 世界模型，用 prediction loss 加 SIGReg 两个简洁的损失项实现了竞争性的控制性能。但在 Two-Room 这类相对简单的环境上，LeWM 的成功率（87%）明显低于一些更朴素的方法（100%），这暗示其表征可能没有最优地反映状态空间的真实结构。

一个合理的假设是：**SIGReg 强制嵌入分布匹配各向同性高斯，对低内在维度的环境（如 Two-Room 的 2 维状态空间）是过度约束**。真实的状态流形被强行撑到高维的高斯形状，可能稀释了状态之间的拓扑关系。

另一个观察是：**球面表征在自监督学习领域已经是被反复验证的成熟选择**。从 SimCLR 到 CLIP 到 DINO，几乎所有现代对比学习方法都把表征归一化到单位球面上。这种选择的优势包括避免高维下的距离集中现象、训练稳定性、和余弦相似度的几何自然性。但球面表征在世界模型场景下的具体效果尚未被系统验证——LeWM 用的是欧氏空间，V-JEPA 2 也是欧氏空间。

本工作提出一个最小化的研究问题：**在世界模型场景下，把欧氏表征 + SIGReg 替换成球面表征 + 简单的对比损失，能否带来改进**？这个问题的答案不仅决定一个具体方法的成败，更重要的是验证一个核心假设——LeJEPA 证明的"各向同性高斯最优"结论是否适用于世界模型场景，还是只适用于静态表征学习场景。

## Background

### LeWorldModel 的核心机制

LeWM 由一个 ViT 编码器 $\text{enc}_\theta: \mathcal{O} \to \mathbb{R}^d$ 和一个 transformer 预测器 $\text{pred}_\phi: \mathbb{R}^d \times \mathcal{A} \to \mathbb{R}^d$ 组成。训练目标是：

$$\mathcal{L}_{\text{LeWM}} = \underbrace{\|\hat{z}_{t+1} - z_{t+1}\|^2}_{\text{预测损失}} + \lambda \cdot \underbrace{\text{SIGReg}(Z)}_{\text{分布形状约束}}$$

预测损失让 predictor 学习动力学。SIGReg 通过 Cramer-Wold 投影把"检验高维分布是否高斯"化归到一维上的 Epps-Pulley 检验，强制嵌入边缘分布匹配各向同性高斯。两个项缺一不可——没有预测损失表征无法反映动力学，没有 SIGReg 表征会塌缩到一点。

LeWM 的设计哲学是用最少的损失项实现稳定训练。整个方案只有一个 trade-off 超参数（SIGReg 的权重 $\lambda$），训练在单 GPU 上几小时内可以完成。

### LeJEPA 的理论框架

LeJEPA 证明在静态表征学习场景下，**各向同性高斯是 JEPA 嵌入分布的最优形式**。证明分两个部分：第一部分证明 covariance 必须是各向同性的（在固定 trace 约束下，这种 covariance 让线性 probe 的 bias 和 variance 都最小）；第二部分证明在固定二阶矩约束下，高斯是最大熵分布，因此让非线性 probe 的 integrated squared bias 最小。

这个定理的关键假设是下游任务是任意线性或核 probe、对所有方向均匀敏感、表征空间是 $\mathbb{R}^d$。这些假设在静态表征学习场景下是合理的（"任意下游任务"是合理的 worst-case），但在世界模型场景下不一定成立——世界模型的下游任务是规划，规划有具体的几何结构（追求"相邻状态有相似表征、远离状态有不同表征"），不是任意的。

LeJEPA 没有覆盖球面表征空间下的最优性问题。Field-JEPA 的论点不是反驳 LeJEPA，而是认为世界模型场景的下游任务结构让另一种几何（球面）成为可行选择，需要实证检验。

### 球面对比学习的成熟经验

球面表征在自监督学习中是被验证的成熟选择。SimCLR、MoCo、CLIP、DINO 等方法都把表征 $L^2$ 归一化到单位球面，使用余弦相似度作为度量，用 alignment（让正样本对齐）和 uniformity（让所有样本均匀分布在球面上）两个目标驱动训练。Wang & Isola 2020 的工作给出了这套框架的理论分析。

球面归一化的几个核心优势：避免高维空间的距离集中现象，让相似度度量在高维下保持区分度；提供天然的尺度约束，避免表征 magnitude 发散；和余弦相似度的兼容性让相似度计算便宜且数值稳定；紧致空间让"覆盖"和"分散"等概念有明确的几何意义。

V-JEPA 2 的潜在表征虽然不是显式的球面归一化，但分析显示它**自然形成 hypersphere pocket 结构**——不同概念以连续的方向变化而不是离散跳跃的形式编码。这暗示 JEPA 类方法可能本来就倾向于学到球面几何，只是没有显式地利用这一点。

## Method

### 核心想法

把 LeWM 的欧氏表征空间 $\mathbb{R}^d$ 替换为球面表征空间 $S^{d-1}$，把 SIGReg 替换为简单的对比形式损失。每个观测 $o$ 通过 encoder 被映射到球面上的一个点 $\mu(o) \in S^{d-1}$，predictor 预测下一时刻的球面点 $\hat{\mu}_{t+1}$。

这个修改保留 LeWM 的所有其他设计——同样的 ViT 编码器骨干、ViT-S 预测器、相同的训练数据、相同的下游 CEM 规划过程。唯一的变化是表征所住的几何空间和防塌缩机制。

### 架构

**Encoder**：保留 LeWM 的 ViT-Tiny 骨干结构。在原本输出 192 维向量的位置加一个 linear 投影到目标维度 $d$，然后做 $L^2$ 归一化得到 $\mu(o) \in S^{d-1}$。这个修改只增加一个线性层和一次归一化操作，其他完全不变。

**Predictor**：保留 LeWM 的 ViT-S 结构（包括动作通过 AdaLN 注入的机制）。输入和输出都从欧氏向量变成球面点——输入 $\mu_t \in S^{d-1}$，输出 $\hat{\mu}_{t+1} \in S^{d-1}$，输出端同样加 $L^2$ 归一化。

球面表征的维度 $d$ 是一个新的超参数，但它的语义和 LeWM 的嵌入维度对应——可以从 16 或类似的较小维度开始。

### 训练目标

总损失包含两项：

$$\mathcal{L} = \mathcal{L}_{\text{pred}} + \lambda \cdot \mathcal{L}_{\text{spread}}$$

**预测损失** 衡量 predictor 输出与下一时刻真实表征的方向差异。最简单的形式是余弦距离：

$$\mathcal{L}_{\text{pred}} = 1 - \cos(\hat{\mu}_{t+1}, \mu_{t+1})$$

由于 $\hat{\mu}$ 和 $\mu$ 都是单位向量，余弦就是它们的内积。最小化这个损失等价于让预测的方向和真实的方向尽可能对齐。

**Spread 损失** 防止表征塌缩。最简单的形式是 batch 内 pairwise 余弦相似度的平均：

$$\mathcal{L}_{\text{spread}} = \frac{1}{B(B-1)} \sum_{i \neq j} \mu_i \cdot \mu_j$$

最小化这个损失鼓励 batch 内不同观测的表征互相分散。它是 Wang & Isola 2020 中 uniformity loss 的一个简化形式，对应"让边缘分布在球面上尽可能 spread out"的目标。

**两项之间的张力**给出方案的核心机制。预测损失要求时间相邻的观测有相似表征（让 predictor 能学到动力学），spread 损失要求 batch 内不同观测的表征互相分散（防止塌缩）。这两个力的平衡点是：**时间相邻的观测对应球面上接近的点，时间远离或不相关的观测对应球面上不同的点**——这正是反映物理状态拓扑的表征。

只有一个 trade-off 超参数 $\lambda$，与 LeWM 的 SIGReg 权重数量相同。

### 实现的最简方案与可能的升级

**预测损失**：最简方案就是上面的余弦距离。这是一个数值稳定、计算便宜、广泛使用的损失形式，没有需要升级的地方。如果发现训练后期收敛缓慢，可以考虑加一个温度参数让梯度更 sharp。

**Spread 损失**：最简方案是 pairwise 余弦相似度的平均。这是一个可计算、可微、有正确方向性的代理。

*如果计算预算允许*：升级到 InfoNCE 形式的 spread 估计器。InfoNCE 对每个 $\mu_i$ 计算它与 batch 内其他 $\mu_j$ 的归一化 softmax，损失是负对数归一化值。这给出一个真正的互信息变分下界，比 pairwise 平均准确。计算成本只是多一个 softmax 操作，可以接受。InfoNCE 在大规模对比学习中是标准做法，超参数选择有大量经验。

另一个值得考虑的升级是用 Wang & Isola 2020 的 uniformity loss 的精确形式：

$$\mathcal{L}_{\text{uniform}} = \log \mathbb{E}[\exp(-t \cdot \|\mu_i - \mu_j\|^2)]$$

这是一个有理论支撑的 uniformity 度量，鼓励样本在球面上均匀分布（在 $t \to \infty$ 的极限下精确测量分布到均匀分布的距离）。

**球面归一化的实现细节**：直接的 $L^2$ normalize 可能在训练早期出现数值不稳定（如果输入向量接近零）。一个简单的修补是在归一化前加一个小常数：$\mu = z / (\|z\| + \epsilon)$，其中 $\epsilon$ 是个很小的数（比如 $10^{-8}$）。这是个标准技巧，没有副作用。

**Predictor 输出的归一化**：predictor 输出后做 $L^2$ normalize，确保 $\hat{\mu}$ 也在球面上。这让预测损失（余弦距离）的计算保持一致。

## Experiment Design

### 整体策略

实验分阶段进行，每个阶段验证一个具体假设。**第一阶段（最小实验 V0）是核心**——它独立验证"球面表征 + 简单对比损失"是否能在世界模型场景下匹配或超过 LeWM 的欧氏表征 + SIGReg。这个阶段的结果决定整个研究方向是否值得继续。

后续阶段是对 V0 的扩展——加入更复杂的元素（vMF 参数化、几何 OOD 边界）来探索是否能进一步改善性能或获得新的能力。后续阶段只在前一阶段成功时进行。

每个阶段的设计原则是**只引入一个变化**，让结果的归因清晰。如果某个阶段的结果显著变化，可以确定是这一步引入的元素的贡献。

### 阶段 1：最小实验 V0

**实现的方案**：上面 Method 部分描述的完整方案——球面 encoder 输出 + 球面 predictor + 余弦预测损失 + pairwise spread 损失。实现工作主要是修改 LeWM 的 encoder 和 predictor 接口、替换损失函数、移除 SIGReg。

**实验环境**：从 Two-Room 开始，因为这是 LeWM 的已知弱点环境（87% 成功率）。如果 V0 在 Two-Room 上能 work，再扩展到 PushT、OGBench-Cube、Reacher 这些 LeWM 的标准环境。

**对比基线**：原版 LeWM。两个版本使用完全相同的训练数据、相同的 architecture 骨干、相同的 CEM 规划设置，只在表征几何和损失函数上有差异。

**评估指标**：

主要指标是 **in-distribution 任务成功率**——和 LeWM 的评估协议完全一致（CEM planning，相同的 sample budget、iterations、horizon）。这是最直接的端到端性能度量，决定 V0 是否有 practical value。

次要指标是 **训练动态的稳定性**——loss 曲线、梯度范数、有效维度的演化。这些指标告诉我们球面表征在世界模型训练中的行为是否健康，是否有 LeWM 没有的训练稳定性问题。

诊断指标包括 **表征几何的可视化**——用 t-SNE 或 UMAP 把训练后的 $\mu(o)$ 投影到 2D，目视检查是否反映物理状态空间的结构。对 Two-Room，预期看到反映两个房间和门连接关系的 2D 流形。对 PushT，预期看到反映 agent 位置和 block 位置的子结构。这些可视化是验证"球面表征是否真的捕捉物理结构"的直接证据。

为了区分 signal 和 noise，每个版本应该跑多个 random seeds（典型是 3-5 个），报告均值和标准差。

**判断标准**：

- 如果 V0 的成功率明显高于或匹配 LeWM（在统计意义上没有劣势），且表征几何诊断显示物理结构得到反映——核心假设成立，进入阶段 2。
- 如果 V0 明显劣于 LeWM——核心假设不成立，停止后续阶段，把这个负面结果作为对 LeJEPA 在世界模型场景的扩展验证。
- 如果 V0 和 LeWM 性能相当但训练更稳定或更简洁——也算 partial success，球面方向有 implementation 优势但没有性能优势。

### 后续阶段的简要规划

**阶段 2：加入 vMF 参数化**。在 V0 基础上让 encoder 同时输出每个观测的集中度参数 $\kappa(o)$，每个观测从球面点变成 vMF 分布 $\rho(o) = \text{vMF}(\mu(o), \kappa(o))$。Predictor 也输出 $(\hat{\mu}, \hat{\kappa})$。预测损失从余弦距离扩展为同时包含方向差异和集中度差异。加入一个 sharp loss 鼓励 $\kappa$ 大（让每个分布集中而不是 spread out）。

这个阶段验证的假设是：**自适应分辨率**（不同观测对应不同确定性的分布）能否带来额外的性能提升。预期在物理状态空间有明显热点的环境上更有用。

**阶段 3：加入 center 约束**。在阶段 2 基础上引入一个可学习的虚拟中心 $\mu_0 \in S^{d-1}$，加一个 hinge loss 把所有训练数据的 $\mu$ 约束在以 $\mu_0$ 为中心、最大角距离为 $\theta_{\max}$ 的球冠内。这给出一个明确的"已知区域"的几何边界，球冠外的球面区域作为 OOD 区域。

这个阶段引入 **OOD 检测能力**作为方案的新特性。除了 in-distribution 性能，还需要新的评估——在 perturbed 数据（视觉变化、物理参数变化）上测试 OOD 检测的 recall 和 precision，并且要验证 in-distribution 性能不被 center 约束损害。

OOD 评估的具体设计包括三类测试：像素级 OOD（修改视觉属性如 agent 颜色），物理 OOD（修改环境参数如门位置），动力学 OOD（应用不可能的动作序列）。每类 OOD 都有不同的预期——像素 OOD 测试表征是否对视觉细节过度敏感，物理 OOD 测试 OOD 边界是否能捕捉训练分布之外的状态，动力学 OOD 测试 prediction error 是否能作为辅助 OOD 信号。

### 阶段之间的递进逻辑

阶段 1、2、3 形成一个 ladder。每个阶段在前一个的基础上只加一个元素，让贡献的归因清晰。

如果阶段 1 失败，stop。如果阶段 2 没有比阶段 1 进一步改善，那 vMF 参数化在这个场景下是不必要的复杂度，论文的核心贡献是 V0。如果阶段 3 没有进一步改善但 OOD 检测能力出现且 in-distribution 性能保持，那 center 约束是有价值的扩展，论文包含 V0 加 OOD 检测的两个独立贡献。

这个 ladder 的优势是**每一个 stop 点都是一个有意义的论文 framing**。即使最终只完成 V0，结果也能作为"球面世界模型"的初步验证发表。即使 vMF 没有改善 V0，"自适应分辨率在简单世界模型环境下的 negative result" 仍然是有信息量的发现。即使 center 约束没有改善 in-distribution 性能但提供了 OOD 检测能力，这本身就是一个有价值的功能扩展。

### 一个执行上的原则

整个实验过程遵循一个原则：**先用最简方案跑通，再决定是否升级**。每一次升级都应该由具体的失败模式触发——发现训练不稳定时考虑升级 spread loss 到 InfoNCE 形式，发现 in-distribution 性能不够时考虑加 vMF 参数化，发现需要 OOD 能力时考虑加 center 约束。不预先做所有升级，因为那会引入不必要的复杂度，让调试变困难。

简单方案失败时的诊断信息更清晰——你能精确知道是哪一层出了问题。复杂方案失败时所有的元素都是嫌疑人，调试成本几何级上升。这种"先简单后复杂"的策略不只是偷懒，它是科学方法论的具体体现——通过控制变量隔离每个设计决策的贡献。

整个研究的核心价值不在于预测某个具体方案一定成功，而在于**无论结果如何都能推进对问题的理解**。如果 V0 成功，它证明球面几何对世界模型表征是有用的，开启了一个新的研究方向。如果 V0 失败，它支持 LeJEPA 的最优性结论在世界模型场景下也成立，这是对已有理论的有意义扩展。两种结果都比一个"跑出数字但不知道为什么"的实验有价值。
